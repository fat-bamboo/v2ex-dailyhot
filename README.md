# V2EX 每日热门话题

V2EX 热议话题榜，展示从 2020-12-18 日开始的 V2EX 热议话题榜单。

## 主要意图

- 让我至少可以看到昨天的热门主题；
- Github 每天多“绿”一下；

## 好处以及缺点

- 不用再看到 8:00 后空空的热门栏 && 可以看到昨天以及之前的热门话题；
- UI 风格更加简约以及突出重点（虽然只有一个标题~）；
- 托管于 Github，所以存在国内网络访问慢甚至访问不了的问题。
- 数据来源于官方 API，所以存在数据不“新”的问题；
